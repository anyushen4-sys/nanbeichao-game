### 统一风格要点
- **墨色基调**：主色 `#1a1410`(焦墨) → `#4a3e32`(淡墨)，部分 `#8b1a2b`(朱砂红) 点缀
- **笔触效果**：所有SVG使用 `feTurbulence+f eDisplacementMap` 滤镜模拟水墨晕染/飞白效果
- **尺寸**：120×120px，保持与现有 `assets/cards/card_*.svg` 一致
- **背景**：透明底（HTML中 `card-illustration` 容器提供阵营底色+渐变）
- **墨渍装饰**：每个SVG角落有1-2个随机半透墨点增加画面趣味

## 实施建议

### 批量命名规则
```
assets/cards/card_{id}.svg
```
`id` 与 `CARDS_DATA` 中的id字段一致（1-126），方便HTML中通过 `card.id` 动态加载。

### HTML引用方式变更（供coder参考）
当前 `ROW_ICONS` 全部使用 `card-hero.png`。建议改为：
```javascript
function getCategoryIcon(card){
  const svgFile = `assets/cards/card_${card.id}.svg`;
  return `<img src="${svgFile}" class="card-svg-icon" style="width:48px;height:56px">`;
}
```
或使用 `use` 引用行对应的SVG作为fallback：
```javascript
function getCategoryIcon(card){
  const custom = `assets/cards/card_${card.id}.svg`;
  const fallback = ROW_ICONS[getCardCategory(card)];
  return `<img src="${custom}" onerror="this.src='${fallback.match(/"([^"]+)"/)[1]}'" class="card-svg-icon">`;
}
```
